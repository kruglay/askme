== ASKME

Учебное приложение, создано по аналогии ask.fm

Изучен паттерн MVC, получены навыки работы с основными механизмами RoR:
    * модели, вьюхи, контроллеры
    * лейауты, формы, хелперы, паршиалы
    * использование локалей(i18n)
    * связи в БД(один к одному, один ко многим, многие ко многим)

Ссылка на деплой(heroku): https://askmelite.herokuapp.com/
